## Filtragem de SNPs

Crie um diretório:

```bash
mkdir pratica2
```

Para avaliarmos a distribuição de dados faltantes por indivíduos (genotype missing data, geno), menor frequência alélica (Minor Allele Frequency, MAF) e desequilíbrio de ligação (Linkage Disequilibrium, LD) ao longo dos dados, utilizaremos o software plink (manual em https://www.cog-genomics.org/plink/2.0/
).

Para baixar o programa plink, use:

```bash
wget https://s3.amazonaws.com/plink1-assets/plink_linux_x86_64_20210416.zip
```

Descompacte o arquivo:

```bash
unzip plink_linux_x86_64_20210416.zip
```

O dataset desta etapa contém dados genômicos da espécie Littoraria flava, um gastrópode marinho que vive nos costões rochosos da costa brasileira. Os dados foram obtidos pela técnica GBS (Genotyping-by-Sequencing), isto é, uma metodologia NGS, e são apresentados aqui no formato .ped e .map. Estes dois arquivos fornecem informações sobre cada indivíduo sequenciado e cada SNP identificado, respectivamente.

Obtenha o dataset:

```bash
scp -P 2205 deadg@143.107.247.17:pratica2/2026/* ./
```

### Questão 01

a. Olhe o sumário do dataset para identificar quantos indivíduos e quantas variantes (SNPs) são contabilizadas:

```bash
./plink --file Lflava --allow-extra-chr
```

Observe o output do comando mostrado no terminal. Quantos SNPs (variants) e quantos indivíduos o arquivo contém?

b. Filtre por MAF (Minor Allele Frequency) utilizando 0.01 como cut-off, o que permitirá somente SNPs com frequência alélica mínima de 1%.

```bash
./plink --file Lflava --allow-extra-chr --maf 0.01 --make-bed
```

Quantos SNPs restaram após essas filtragens?

c. Faça filtragens utilizando o valor de corte 0.05 e responda: Tendo em vista a natureza do dataset e seu uso em análises de estruturação populacional, qual cut-off você utilizaria? Explique com base na relação entre estruturação genética e variantes raros (MAFs <0.01 e 0.05).

d. Filtre por LD por meio da taxa r², a qual será calculada a cada conjunto de 50 SNPs com um intervalo de 5pb entre eles, removendo os SNPs com r² > 0.5. Lembre-se que o valor r² varia de 0 (sem associação/herança independente) a 1 (correlação total entre alelos).

```bash
./plink --file Lflava --allow-extra-chr --indep-pairwise 50 5 0.5 --make-bed
```

Explique a importância da remoção dos SNPs que se encontram sob desequilíbrio de ligação e como a não remoção destes poderia interferir em análises de diferentes tipos, como as populacionais, por exemplo. Houve alta porcentagem de SNPs sob desequilíbrio de ligação?

e. Filtre por missing data, permitindo no máximo de 10% de missing por SNP. Qual foi a taxa de remoção de missing data e o que isso nos diz sobre a qualidade dos dados?

```bash
./plink --file Lflava --allow-extra-chr --geno 0.1 --make-bed
```

f. Filtre os dados, simultaneamente, combinando diferentes cut-offs de por MAF, LD e missing data, gerando o vcf com o subdataset ao final:

```bash

```
